# 音乐后台管理系统 java302

## 项目概述

音乐后台管理系统是一款基于SpringBoot和Vue的前后端分离式管理系统。它提供了一套完整的音乐后台解决方案，涵盖了音乐数据的管理和操作。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 系统环境
- 兼容Windows和macOS操作系统

### 推荐运行环境
- jdk1.8
- maven
- MySQL
- nodejs

### 服务端技术
- Springboot
- Mybatis-Plus

### 前端技术
- Vue
- axios
- Element-UI

## 功能模块

具体功能模块如下：

1. 音乐数据管理
2. 用户权限控制
3. 音乐上传与下载
4. 音乐分类管理
5. 播放列表管理

## 系统角色

- 系统管理员
- 普通用户

## 运行环境

- 服务端：Springboot应用程序
- 客户端：Vue前端应用程序

## 部署步骤

1. 准备推荐运行环境
2. 导入数据库文件
3. 编译并运行后端服务
4. 编译并运行前端服务
